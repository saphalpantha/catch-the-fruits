# Catch the Fruits

A simple browser-based arcade game where players catch falling fruits to score points. Built with vanilla JavaScript, HTML, and CSS, this project focuses on core game logic, animations, and responsive gameplay.

---

## Description

Catch the Fruits is a lightweight web game developed using plain JavaScript without external frameworks. The player moves a basket to catch falling fruits and earn points, with increasing difficulty over time. ## Features

- Core game loop built with JavaScript
